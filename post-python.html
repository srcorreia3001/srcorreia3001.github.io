Análise de Dados com Python: Caso Unisanta

Postado em: 26 de Fevereiro de 2026 | Categoria: Python / Ciência de Dados

Neste projeto, explorei o tratamento de dados utilizando a biblioteca Pandas. O objetivo principal foi identificar padrões de comportamento em uma base de dados bruta.

O Desafio Técnico

O maior problema encontrado foi o alto número de dados nulos e formatos inconsistentes. Abaixo, mostro como resolvi isso usando Python:

import pandas as pd

# Carregando os dados
df = pd.read_csv('dados_originais.csv')

# Limpando valores nulos
df_clean = df.dropna(subset=['id_cliente'])

# Gerando estatísticas
print(df_clean.describe())

Com essa limpeza, a precisão do modelo final subiu de 65% para 88%.
