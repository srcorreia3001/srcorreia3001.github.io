Consultas Complexas e Otimização SQL

Como futuro Analista de Dados, o SQL é minha ferramenta diária para extração de KPIs.

SELECT cliente, SUM(valor_venda) AS total
FROM vendas
WHERE data_venda >= '2026-01-01'
GROUP BY cliente
ORDER BY total DESC;

Este comando permite gerar um ranking dos melhores clientes do trimestre de forma instantânea.
