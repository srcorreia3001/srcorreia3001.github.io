Gestão de Escola Bíblica Dominical (EBD)

Status: Em Desenvolvimento (Projeto Real)
Tecnologias: Python, SQLite, Lógica de Programação
Instituição: Universidade Santa Cecília (UniSanta)

Descrição do Projeto

Este projeto nasceu da necessidade real de organizar e digitalizar o controle de uma Escola Bíblica Dominical. Como estudante de Ciência de Dados, apliquei conceitos de estruturação de dados para criar uma ferramenta capaz de gerenciar o cadastro de alunos, professores e o registro de frequências semanais.

Funcionalidades Técnicas

A aplicação foi construída utilizando a linguagem Python, focando em uma interface funcional e um back-end robusto para evitar a perda de dados. O sistema permite gerar relatórios de assiduidade, auxiliando a secretaria da igreja na tomada de decisões baseada em dados reais de participação.

Objetivo Acadêmico

Desenvolver este software me permite praticar a manipulação de bibliotecas Python e a integração com bancos de dados, habilidades fundamentais para minha transição de carreira para a área de análise e ciência de dados.
